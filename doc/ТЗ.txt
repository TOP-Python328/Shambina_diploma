1. Введение
1.1. Наименование веб-портала: "Физика на инженерно-физическом факультете Адыгейского государственного факультета"
1.2. Портал "Физика на ИФФ АГУ" предназначен для абитуриентов и школьников, интересующихся физикой.
Цель - заинтересовать физикой, рассказать об ИФФ АГУ, помочь с подготовкой к поступлению на кафедру теоретической физики ИФФ АГУ.
Для интересующихся физикой портал предоставляет описание опытов по физике, материалы мастер-классы по конструированию различных устройств, новости науки и полезные ссылки.
Для тех, кто готовится ко вступительным экзаменам, портал предоставляет онлайн-обучение для подготовки к ОГЭ, к ЕГЭ.
В разделе "Консультации" описываются часто встречающиеся ошибки и способ их избежать.
На форуме зарегистрированный пользователь может задать вопрос преподавателю и получить ответ,
пользователи могут общаться между собой, преподаватели тоже могут общаться между собой.
Кроме того, предоставляются задания вступительных экзаменов на ИФФ АГУ прошлых лет и полезные ссылки для подготовки к экзаменам.
Для тех, кто выбирает ВУЗ, портал рассказывает о факультете, предоставляет организационную информацию о поступлении и рассказывает новости ИФФ АГУ.
Кроме того, на форуме можно задать вопрос организационного плана о процессе поступления на ИФФ АГУ.
2. Краткое название веб-портала: "Физика на ИФФ АГУ"
3. Раздел "Опыты по физике"
- Функциональное назначение - в разделе отображается меню - список опытов по физике, являющихся ссылками на страницы с опытами.
Меню выглядит как фотография опыта, его название и раздел физики, к которому относится опыт, рейтинг популярности.
Есть возможность отсортировать опыты по алфавиту, по разделам физики, по популярности.
На странице каждого опыта отображается название опыта, список оборудования, видеоролик его выполнения, текстовый комментарий с фотографиями пошагового выполнения, физическое объяснение (возможно с формулами).
Все указанные данные по каждому опыту можно скачать в файле pdf для печати.
Внизу страницы есть обсуждение опыта, где зарегистрированные пользователи могут оставить сообщение.
Кроме того, на странице опыта можно поставить ему лайк или дизлайк. Процентное соотношение лайков и дизлайков определяет популярность опыта.
У зарегистрированных пользователей есть возможность предложить опыт в виде его описания и, по желанию - видео.
- Эксплуатационное назначение - раздел помогает школьникам лучше понять раздел физики, заинтересовывает, описывает, как повторить этот опыт дома.
В профиле зарегистрированного пользователя отображается список опытов (со ссылками на них), которым он поставил лайк.

4. Раздел "Конструкторское бюро"
- Функциональное назначение - предоставление материалов мастер-классов по конструированию различных устройств в формате аналогичном разделу "Опыты по физике".
- Эксплуатационное назначение - пользователи сайта смогут ознакомиться с процессом создания разнообразных устройств и создать их самостоятельно, опираясь на предоставленные материалы.
Это способствует повышению мотивации к изучению физики, а созданные устройства будут полезны в повседневной жизни.

5. Раздел "Новости науки"
- Функциональное назначение - освещение новых разработок на базе физических знаний, описание передового края науки и прогресса развития науки.
Раздел устроен аналогично разделу "Опыты по физике"
- Эксплуатационное назначение - пользователи могут познакомиться с современными областями применения физических знаний и узнать новости науки.

6. Раздел "Полезные ссылки"
- Функциональное назначение - предоставление описания ссылок и самих ссылок, ссылки сгруппированы по разделам.
- Эксплуатационное назначение - пользователь сможет быстро найти полезные сайты на интересующую его тематику, связанную с физикой.

7. Раздел "Подготовка к ОГЭ"
- Функциональное назначение
Подготовка к ОГЭ это квест, где нужно пройти несколько глобальных заданий (уровней). Чтобы пройти глобальное задание, нужно пройти несколько локальных заданий. Локальные задания идут в таком порядке:
1) видео с материалами, сразу под видео конспект;
2) список простых вопросов с тестами;
3) список простых заданий;
4) задания посложнее;
5) возможно повторение первых четырёх пунктов.
6) В конце списка локальных заданий - домашнее задание, которое нужно выполнить и прислать на проверку учителю.
В конце каждого задания 1)-6) можно задать вопрос, на который могут ответить и преподаватели и учащиеся.
Все виды локальных заданий, кроме домашнего задания, проверяются автоматически.
Каждый пользователь получает зачёт/незачёт, проходя задания. Пока он не получит зачёт по предыдущему заданию (любого уровня вложенности), он не сможет перейти к следующему.
В конце курса ученик получает сертификат о прохождении курса.

8. Раздел "Подготовка к ЕГЭ"
Осуществляется также, только относительно ЕГЭ.

9. Раздел "Часто встречающиеся ошибки"
- Функциональное назначение - предоставление информации о том, какие ошибки чаще всего встречаются среди решений ОГЭ или ЕГЭ, и о том, как их избежать. Раздел содержит ссылки на сгруппированные по темам ошибки. Пройдя по ссылке, пользователь может посмотреть видео с объяснением ошибки, как её избежать, и конспект. Возможны примеры задач, в которых может встретиться данная ошибка.

10. Раздел "Задать вопрос преподавателю"
- Функциональное назначение - дать возможность ученикам уточнить непонятные моменты у преподавателя.
