1. Введение
1.1. Наименование веб-портала: "Физика на инженерно-физическом факультете Адыгейского государственного факультета"
1.2. Портал "Физика на ИФФ АГУ" предназначен для абитуриентов и школьников, интересующихся физикой.
Цель - заинтересовать физикой, рассказать об ИФФ АГУ, помочь с подготовкой к поступлению на кафедру теоретической физики ИФФ АГУ.
Для интересующихся физикой портал предоставляет описание опытов по физике, материалы мастер-классы по конструированию различных устройств, новости науки и полезные ссылки.
Для тех, кто готовится ко вступительным экзаменам, портал предоставляет онлайн-обучение для подготовки к ОГЭ, к ЕГЭ.
В разделе "Консультации" описываются часто встречающиеся ошибки и способ их избежать.
На форуме зарегистрированный пользователь может задать вопрос преподавателю и получить ответ,
пользователи могут общаться между собой, преподаватели тоже могут общаться между собой.
Кроме того, предоставляются задания вступительных экзаменов на ИФФ АГУ прошлых лет и полезные ссылки для подготовки к экзаменам.
Для тех, кто выбирает ВУЗ, портал рассказывает о факультете, предоставляет организационную информацию о поступлении и рассказывает новости ИФФ АГУ.
Кроме того, на форуме можно задать вопрос организационного плана о процессе поступления на ИФФ АГУ.
2. Краткое название веб-портала: "Физика на ИФФ АГУ"
3. Раздел "Опыты по физике"
- Функциональное назначение - в разделе отображается меню - список опытов по физике, являющихся ссылками на страницы с опытами.
Меню выглядит как фотография опыта, его название и раздел физики, к которому относится опыт, рейтинг популярности.
Есть возможность отсортировать опыты по алфавиту, по разделам физики, по популярности.
На странице каждого опыта отображается название опыта, список оборудования, видеоролик его выполнения, текстовый комментарий с фотографиями пошагового выполнения, физическое объяснение (возможно с формулами).
Внизу страницы есть обсуждение опыта, где зарегистрированные пользователи могут оставить сообщение.
Кроме того, на странице опыта можно поставить ему лайк или дизлайк. Процентное соотношение лайков и дизлайков определяет популярность опыта.
У зарегистрированных пользователей есть возможность предложить опыт в виде его описания и, по желанию - видео.
- Эксплуатационное назначение - раздел помогает школьникам лучше понять раздел физики, заинтересовывает, описывает, как повторить этот опыт дома.
В профиле зарегистрированного пользователя отображается список опытов (со ссылками на них), которым он поставил лайк.
